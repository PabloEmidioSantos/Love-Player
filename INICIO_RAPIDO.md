# 🚀 INÍCIO RÁPIDO

## Para Windows:

```bash
# 1. Abrir Power Shell ou CMD

# 2. Navegar para a pasta do projeto
cd c:\Users\pablo.santos\Desktop\pablo_pessoal\EstudosGeral\python\Love-Player

# 3. Criar ambiente virtual
python -m venv venv

# 4. Ativar ambiente virtual
venv\Scripts\activate

# Você deve ver (venv) no início da linha do terminal

# 5. Instalar dependências
pip install -r requirements.txt

# 6. Executar servidor
python app.py

# 7. Abrir navegador e acessar
# http://localhost:1905
```

## Para Linux/Mac:

```bash
# 1. Abrir terminal

# 2. Criar ambiente virtual
python3 -m venv venv

# 3. Ativar
source venv/bin/activate

# 4. Instalar
pip install -r requirements.txt

# 5. Executar
python app.py

# 6. Acessar http://localhost:1905
```

## ⚡ Se der erro:

```bash
# Erro: "python não reconhecido"
# Solução: Use "python3" ao invés de "python"

# Erro: "Module not found"
# Solução: Verifique se o ambiente virtual está ativado
# Você deve ver (venv) no início da linha

# Erro: "Port 1905 already in use"
# Solução: Mude a porta no arquivo .env
```
