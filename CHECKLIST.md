## 🚀 Como Começar (3 PASSOS)

### PASSO 1: Ambiente Virtual
```bash
python -m venv venv
venv\Scripts\activate  # Windows
# ou
source venv/bin/activate  # Linux/Mac
```

### PASSO 2: Instalar Dependências
```bash
pip install -r requirements.txt
```

### PASSO 3: Executar
```bash
python app.py
# Abra: http://localhost:1905
```

## 🏗️ Arquitetura

### Padrão MVC Profissional

```
Frontend (HTML/CSS/JS)
    ↓ fetch()
Rotas (rotas.py)
    ↓ chama
Serviço (servico_musicas.py)
    ↓ usa
Modelos (modelos.py)
    ↓ retorna JSON
Frontend renderiza
```

## 🔌 API REST Completa

```
GET /                      → Página inicial
GET /api/musicas           → Todas as 21 músicas
GET /api/musicas/<id>      → Música específica
GET /api/musicas/<id>/proxima    → Próxima música
GET /api/musicas/<id>/anterior   → Música anterior
GET /api/playlist/info     → Info da playlist
GET /api/health            → Health check
```
